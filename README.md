# 提詞機・速成倉頡字典

呢個專案係一個單頁應用（可離線/PWA），主要功能包括：
- 速成／倉頡查字
- 提詞機／打字練習
- **粵語同音字查詢（輸入中文 → 顯示對應粵拼＋同音字）**

## 更新內容（2026-06-27）
- **已完全移除「發聲/朗讀」功能**（避免裝置兼容/網絡限制造成困擾）。
- **同音字候選排序改為「常用字優先」**：
  - 粵語口語常用字（例如：`佢、嘅、唔、喺…`）排前
  - 其餘按字頻表 `src/char_freq_rank.json` 排序（常用→冷門）
- **查單字新增「關聯詞語（常用）」**：查到某個字後，會顯示常見組詞/詞語，點一下可複製。
- **查單字新增「在線增強」**（有網就用）：補充普通話拼音、異體字、普通話同音字、更多常用詞組（冇網照常用離線功能）。
- **PWA 快取已升級至 `cangjie-v20`**（確保已安裝到主畫面嘅版本可以更新）。

## 開發

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## 回歸測試（同音字）

```bash
pnpm test:homophone
```

## PWA / 離線更新提示
如果你之前已經「加入主畫面」安裝過：
1. 重新整理一次（或關閉再開）
2. 如仍然見到舊版：刪除舊版 PWA，再重新加入主畫面
