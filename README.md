# 提詞機・速成倉頡字典

呢個專案係一個單頁應用（可離線/PWA），主要功能包括：
- 速成／倉頡查字
- 提詞機／打字練習
- **粵語同音字查詢（輸入中文 → 顯示對應粵拼＋同音字）**

## 重要修正（2026-06-27）
- 修正「同音字」面板：輸入中文（例如：`我`）會出現「無讀音／此字未收錄／未找到」嘅問題。
  - 原因係中文字判斷（isChinese）喺部分情況會誤判，令系統當你冇輸入中文。
  - 同時加入字典反向索引保護（如有需要會由 `pinyin_to_chars` 自動建立 `char_to_pinyin`）。
- **PWA 快取已升級至 `cangjie-v13`**，確保已安裝到主畫面嘅版本可以更新到最新檔案。
- **粵語朗讀（🔊）加入「網上後備」**：
  - 自動模式：本機有 `zh-HK / 粵語 voice` 就用本機；冇就（有網先）改用網上朗讀
  - 可喺「⚙ 發音設定」入面揀：自動／只用本機／只用網上
- **同音字候選排序改為「常用字優先」**（口語常用字 + 字頻表 `src/char_freq_rank.json`）

## 開發

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## 回歸測試（同音字）

```bash
pnpm test:homophone
```

## PWA / 離線更新提示
如果你之前已經「加入主畫面」安裝過：
1. 重新整理一次（或關閉再開）
2. 如仍然見到舊版：刪除舊版 PWA，再重新加入主畫面
