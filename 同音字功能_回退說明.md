# 粵語同音字功能—使用方法、排序邏輯 & 回退

## 用法
1. 去「查單字」頁（你平時查倉頡／速成嗰頁）。
2. 右下角有個 🔍（同音字查詢），撳佢會彈出「粵語同音字」面板。
3. 喺輸入框打 **中文字**（可以多字，例如：`我哋`）。
4. 下面會按「每個字」分組，列出：
   - 佢嘅粵拼（可能有多個讀音）
   - 每個讀音對應嘅同音字
5. 撳任何一個同音字會：
   - 自動帶去「查單字」輸入格
   - 同時寫入「最近查過」歷史
   - 關閉同音字面板

## 排序（常用字優先）
同音字候選而家會做「常用度排序」：
- **粵語口語常用字**（例如：`佢、嘅、唔、喺、啲、咗…`）會優先排前
- 其餘字會參考 `src/char_freq_rank.json` 嘅字頻排名（常用排前、冷門排後）

> 註：字頻表只係用嚟排序，唔會影響你「搵唔搵到」嗰個字。

## 自檢 / Debug
- 面板內有「自檢同音」按鈕：
  - 會喺 Console 輸出 `🧪 自檢：粵語同音字` 表格
- 正常查詢時，Console 會有：
  - `console.group('[Homophone] query')`

## 離線 / 部署注意
同音字功能需要以下檔案一齊部署（保持相對路徑唔變）：
- `index.html`
- `sw.js`
- `src/hk-homophones.js`
- `src/jyutping_dict.json`
- `src/char_freq_rank.json`

建議用：
- `pnpm dev` / `pnpm preview`（HTTP server）
- 或「加入主畫面」安裝做 PWA

> 如用 `file://` 直接開 `index.html`，部分瀏覽器會封鎖 `fetch()`，有機會令同音字資料載入失敗。

## 回退（還原舊版）
同一個目錄已經放咗舊版：
- `index.restore-20260626_before-homophones.html`

### 方法 A：手動改名
1. 將而家嘅 `index.html` 改名做 `index.new.html`
2. 將 `index.restore-20260626_before-homophones.html` 改名做 `index.html`

### 方法 B：命令列（Windows / macOS / Linux 都類似）
```sh
# 先備份
cp index.html index.new.html
# 還原
cp index.restore-20260626_before-homophones.html index.html
```
