# 粵語同音字功能—使用方法、排序 

## 用法
1. 去「查單字」頁。
2. 右下角 🔍（同音字查詢）→ 會彈出「粵語同音字」面板。
3. 輸入 **中文字**（可多字，例如：`我哋`），或直接輸入粵拼（例：`ngo` / `ngo5`）。
4. 會按「每個字」分組，列出粵拼 + 同音字。
5. 撳任何同音字會：帶去「查單字」輸入格 + 寫入「最近查過」。

## 排序（常用字排前）
同音字候選會做「常用度排序」：
- 粵語口語常用字優先
- 其餘按 `src/char_freq_rank.json` 字頻排序（常用→冷門）

## 離線 / 部署注意
同音字功能需要連同以下檔案一齊部署（保持相對路徑）：
- `index.html`
- `sw.js`
- `src/hk-homophones.js`
- `src/jyutping_dict.json`
- `src/char_freq_rank.json`

> 如用 `file://` 直接開，部分瀏覽器會限制 `fetch()`，建議用 HTTP server（例如 `pnpm dev` / `pnpm preview`）。

## localStorage
相關 key：
- `hk_jyutping_dict_cache_v1`
- `hk_char_freq_rank_cache_v1`
