# 粵語同音字功能（NEW v20260626）— 使用方法 & 回退

## 用法
1. 去「查字」頁（即係你平時查倉頡/速成嗰頁）。
2. 右下角有個 🔍 按鈕（同音字查詢），撳佢會彈出「粵語同音字」面板。
3. 喺輸入框打 **中文字**（可以多字，例如：`我哋`）。
4. 下面會按「每個字」分組，列出：
   - 佢嘅粵拼（可能有多個讀音）
   - 每個讀音對應嘅同音字
5. 你撳任何一個同音字，會：
   - 自動帶去「查字」輸入格
   - 同時寫入「曾查單字」歷史
   - 關閉同音字面板

## 自檢 / Debug
- 面板內有「自檢同音」按鈕：
  - 會喺 Console 輸出 `🧪 自檢：粵語同音字` 表格
- 正常查詢時，Console 會有一組 log：
  - `console.group('[Homophone] query')`

## 離線支援
- 依家同音字字典已經**內嵌**入 `index.html`，唔再靠 `fetch('src/jyutping_dict.json')`。
- 即使你用 `file://` 直接打開都應該可以用。

## localStorage
- 今次同音字功能 **冇新增** 任何 localStorage key。

## 回退（還原舊版）
同一個目錄已經放咗舊版：
- `index.restore-20260626_before-homophones.html`

### 方法 A：手動改名
1. 將而家嘅 `index.html` 改名做 `index.new.html`
2. 將 `index.restore-20260626_before-homophones.html` 改名做 `index.html`

### 方法 B：命令列（Windows / macOS / Linux 都類似）
```sh
# 先備份
cp index.html index.new.html
# 還原
cp index.restore-20260626_before-homophones.html index.html
```
